# Phase 3 — UX Polish

## Goal

Improve clarity without hiding power. Make Studio feel professional and trustworthy.

## Deliverables

1. **Better empty states** — Every page/widget has a helpful empty state (not blank)
2. **Command palette refinement** — Cmd+K search works across all data sources
3. **Optional voice/TTS** — Browser SpeechRecognition for input, Web Speech API or OpenClaw TTS for output
4. **Dashboard layout persistence** — Save widget positions/sizes to localStorage
5. **Drag-and-drop dashboard** — react-grid-layout for widget repositioning
6. **Keyboard shortcut completeness** — Every major action has a keyboard shortcut
7. **Responsive design audit** — All pages work on tablet (1024px) and large monitors (2560px)

## Exit Criteria

- [ ] No blank pages or empty screens without guidance
- [ ] Dashboard layout persists across page reloads
- [ ] Voice input works in at least one browser (Chrome)
- [ ] All keyboard shortcuts documented in ? modal
